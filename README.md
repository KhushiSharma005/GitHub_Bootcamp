# Bootstrap Login Page

A simple and responsive Login Page built using **HTML**, **Bootstrap 5**, and a background image card layout.

## Features

* Responsive design using Bootstrap 5
* Card-based login form
* Background image with overlay content
* Email and Password input fields
* Full-width Login button
* Clean and modern UI

## Technologies Used

* HTML5
* Bootstrap 5.3.8
* CSS (Bootstrap Utility Classes)

## Project Structure

```text
project-folder/
│
├── index.html
├── Login_page.jpg
└── README.md
```

## How to Run

1. Download or clone this repository.
2. Make sure the image file `Login_page.jpg` is in the project folder.
3. Open `index.html` in any web browser.

## Preview

The login page displays:

* A background image inside a Bootstrap card
* Email input field
* Password input field
* Login button
